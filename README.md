# Writer's Revenge

A distraction-free writing app designed to help writers finish what they start.

## Features

- Drafting Mode (no deletion allowed)
- Session-based writing
- Word count tracking
- Emoji-based reflection
- Auto session start

## Tech Stack

- React Native (Expo)
- TypeScript
- Zustand (state management)

## Why I Built This

I struggled with finishing stories due to over-editing and losing momentum.
This app is designed to remove friction and force progress.

## Status

In active development — currently implementing persistence and streak tracking.
